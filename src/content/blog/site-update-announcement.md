个人博客的第一版已经准备好上线。

这个版本先把最重要的部分搭起来：能写文章，能阅读，能搜索，也能随着每次推送自动更新。它不是一个一次性完成的作品，更像一个可以慢慢生长的写作空间。

## 本次更新

- 完成首页、文章列表、文章详情、标签页、关于页和 404 页面
- 支持 Markdown 写作，并按发布时间自动排序
- 加入站内搜索、RSS、站点地图和 robots 配置
- 增加文章封面图、标签、精选文章和基础 SEO 信息
- 配置 GitHub Pages 自动部署流程

## 后续计划

接下来可以继续补充真实文章内容，逐步完善关于页、个人链接、评论区和更细致的视觉风格。

先让它稳定地存在，再让它越来越像自己。
